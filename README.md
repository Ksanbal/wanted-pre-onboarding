## Description

[원티드 프리온보딩 백엔드](https://www.wanted.co.kr/events/pre_ob_be_4) 4차 선발과제

## Installation

```bash
$ yarn
```

## Seeding Database

```bash
# wanted_pre_onboarding.db 파일이 없는 경우에만 실행
$ yarn seed:run
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
